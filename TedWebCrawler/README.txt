* 사전 작업

- Python 3.8 (64-bit)가 설치되어 있어야 합니다. (설치파일 같이 제공)

- 파이썬 스크립트로 selenium 및 bs4 가 설치되어 있어야 합니다.
- 기본적으로 스크립트 경로는 C:\Users\(사용자계정이름)\AppData\Local\Programs\Python\Python38\Scripts에 있습니다.
- cmd창을 관리자모드로 실행 후 순서대로 입력합니다.
  (cd C:\Users\(사용자계정이름)\AppData\Local\Programs\Python\Python38\Scripts)
  (pip install --upgrade pip)
  (pip install bs4)
  (pip install selenium)

- C드라이브 바로 아래에 "result"라는 이름의 폴더를 만들어주세요.

- 실행파일 경로에 chromedriver가 있어야 합니다.
- 이미 같이 제공하게 되어 있습니다. (크롬83버전입니다.)
- 실행이 안된다면 크롬 버전을 확인한 후 버전에 맞는 chromedriver를 설치 후에 실행파일 경로에 넣어 주세요.

----------------------------------------------------------------------------------------------------
주의사항

- C드라이브에 "result"라는 이름의 폴더가 존재해야 하며, 이곳에 전부 저장됩니다.

- 시간초는 5자리까지만 출력됩니다.

- 각 에러 파일은 내용에 에러로 추정되는 링크를 포함합니다.
- titleNameErrorFile : 제목을 제대로 가져오지 못한 파일
- linkErrorFile : 링크가 존재하지 않는 파일
- timeErrorFile : 시간초를 제대로 가져오지 못한 파일
- 에러 파일이나 creator파일 앞에 있는 숫자는 정렬을 위해 넣은 것이므로 아무 의미가 없습니다.

*** 각 페이지마다 2초의 sleep타임이 걸려 있습니다. 시간초나 제목 등에 대한 데이터를 정확히 로딩받기 위함입니다.
     이에 따라, 네트워크 상태가 좋지 않음 등의 이유로 로딩 시간이 길어지면 원하지 않는 결과를 발생시킬 수도 있습니다.
